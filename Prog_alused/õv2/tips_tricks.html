Tips and tricks

global see võtmesõna muudab muutuja funktsiooni sees kasutatavaks väljaspool seeda funktsiooni (muudab ka juba globaalset samanimelist muutujat).

x = "awesome"

print("Python is " + x)  # = Python is awesome

def myfunc(): print("Python is " + x)  # = Python is awesome

def myfunc1(): global x
x = "great"
print("Python is " + x)  # = Python is great

def myfunc2(): print("Python is " + x)  # = Python is great

def myfunc2(): x = "ok"
print("Python is " + x)  # = Python is ok

print("Python is " + x)  # = Python is great
