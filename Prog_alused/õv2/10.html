Ennik

Ennik (tuple) on järjestatud muutumatu andmehulk. Seda saab kasutada sarnaselt listiga (muutmisoperatsioone pole). Kasutatakse Pythonis funktsioonidest mitme väärtuse tagastamiseks.

def example(): return 1, 2, 4

Enniku loomine

Enniku saab luua kasutades tavalisi sulge (). Saab luua ka sulgudeta kuid loetavuse mõttes mõistlik siiski teha sulgudega.

elements = (1, 2, 3)
empty_tuple = ()
one_element_tuple = (1, )

elements = 1, 2, 3
one_element = 1,  # use of comma!

Ennik ja järjend

Ennikut saaab muuta järjendiks kasutades list() korraldust. Järjendit saab muuta ennikuks kasutades tuple() korraldust.

Operatsioonid ennikuga

  • ennik[0] elemendi lugemine indeksi abil.
  • ennik[:3] lõik ennikust (töötab sarnaselt järjendi lõigule).
  • 3 in ennik elemendi esinemine ennikus.
  • tuple("hello") => ('H', 'e', 'l', 'l', 'o') sõne ennikus.
  • (1, 2) + (3,) => (1, 2, 3) enniku liitmine.

Enniku kasutus

  • Ennikut kasutatakse väärtuste pakkimiseks.

elements = 1, 2, 3
a, b, c = elements
a, b, c = (4, 5, 6)

  • Sarnaselt töötab näiteks enumerate ja dict.items().
  • Funktsiooni väärtused saab lahti pakkida.

def hello(): return "Hello", "Tom"
greeting, name = hello()  # "Hello", "Tom"

  • Kuna ennik on muutumatu (immutable) andmetüüp, saab seda kasutada sõnastiku võtmena (järjend ei sobi).
