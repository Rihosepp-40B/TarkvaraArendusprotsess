Sõne

Mis on sõne (string)?

Sõne (str) on andmetüüp:

  • Sõne on andmete sümbolite jada mis edastatakse tekstilisel kujul
  • Sõne on muutumatu objekt (immutable)
    • see tähendab, et sõne ei saa muuta
    • sõne operatsioonid tagastavad vajadusel uue sõne

Sõne loomine

  • Sõne loomiseks kasutatakse ülakomasid, jutumärke või kolmekordseid jutummärke
    • 'See on sõne'
    • "See on sõne"
    • """See on sõne"""
  • Kolmekordsed jutumärgid võimaldavad esitada mitmerealist teksti
    • Kasutades /n reavahetuse kohal saab seda teha ka ülakomade ja jutumärkidega
    • print('See on esimene rida\nSee on teine rida')
    • print("""See on esimene rida
      See on teine rida""")
  • Ülakoma saab kasutada osa sõnena kui sõne tehakse jutumärkide või kolmekordsete jutumärkide vahele või siis varjestatult \'
  • Jutumärke saab kasutada kolmekordsete ja ülakomade vahel või varjestatult \"
  • Sõnesid saab liita + märgiga: text = "Hello " + "world"  # Hello world
  • Sõnet saab korrata * märgiga: text = "hi" * 3  # hihihi

Sõne operatsioonid

Kuna sõne on sümbolite jada siis saame sellele rakendada järjenevaid operatsioone.

greeting = "Hello"
print(len(greeting))  # 5
print(greeting[1])  # e

# slice [start:end], start inclusive, end exclusive
print(greeting[1:3])  # el

  • Kantsulgudega saab määrata, millist elementi küsime
  • Indeksid algavad 0-st
  • Lõik [algus:lõpp] - algus kaasa arvatud, lõpp mitte

Sõne lõiked

  • Lõikes saab määrata alguse, lõpu ja sammu [start:end:step]:
  • Vaikeväärtused: [0:len(text):1]
  • [-1] negatiivne arv leiab tagantpoolt lugedes sellel kohal asuva sümboli (-0 pole olemas, seega -1 on viimane sümbol)
  • [::-1] negatiivne samm pöörab sõne tagurpidi

greeting = "Hello!"
print(greeting[-1])  # !

print(greeting[-3:])  # lo!
print(greeting[:-2])  # Hell

print(greeting[::-1])  # !olleH
print(greeting[-1:1:-2])  # !l

Veel sõne operatsioone

  • .lower() / .upper() esitab sõne väikeste / suurte tähtedega
  • .startswith() annab tõeväärtuse sõne kohta vastavalt sisestatud tingimusele
  • .index() annab sisestatud "otsitava" väärtuse asukoha sõnes
  • .count() loeb kokku otsitava väärtuse kordumise sõnes
  • .replace() asendab sõnes otsitava väärtuse uue väärtusega

Sõne operatsioonid loovad vajadusel uue sõne. Näiteks .replace() asendab tähe, aga teeb selle jaoks uue sõne. Selleks, et algset muutujat "muuta", kirjutame loodud sõne algsesse muutujasse tagasi

greeting = "Hello!"

greeting.replace("e", "a")  # new string is created
print(greeting)  # Hello! <- the old one is unmodified

# to "modify" the string, we need to replace the variable
greeting = greeting.replace("e", "a")
print(greeting)  # Hallo!

Sõne vormindamine ehk f-sõne

Alates Python versioonist 3.6 saab kasutada f-sõne

  • Sõnele märgitakse f täht ette: f"text"
  • f-sõne sees saab loogeliste sulgude vahel kasutada avaldisi
  • f"sum: {1 + 3}" => "sum: 4"
  • Loogeliste sulgude vahel võib kasutada ka muutujaid
  • f"name: {name}" => "name: Ago"
  • Varasemalt liites sõnale arvu, tuli see muuta sõneks, f-sõne teeb seda automaatselt

    • Vormindamine

  • F-sõne saab kasutada vormindamiseks:
    • f"{nr:04}" => "0012"
    • nr on muutuja
    • 0 näitab mida lisme muutuja ette
    • 4 tähistab tulemuse pikkust

Joondamine

Joondamiseks kasutame peale koolonit joondamise suuna tähistamiseks: paremale >, keskele ^, vasakule <. Peale suunda märgitakse laius, mille ulatuses joondamine toimub: f"|{name:>10}|{students:^10}|{title:<10}|"
