Järjend

Mis on järjend?

Järjend (list) on andmetüüp, mis sisaldab elemente. Need võivad olla ükskõik mis andmetüüpi, ka list. Elementide järjekord on fikseeritud. Järjendiga saab: elemente lisada, elemente eemaldada, elemetide väärtust muuta.

Pythonis võib ühes listis olla erinevat tüüpi andmeid. Koodi puhtuse seisukohast pole see soovitatav. Üldiselt on mõistlik listis hoida sama tüüpi elemente.

Järjendi loomine

Järjendi loomiseks määrame muutuja nime ja omistame sellele loendi. Loendit kujutatakse kantsulgudega [].

empty_list = []
numbers = [1, 2, 3, 4]
names = ["Ago", "Kati", "Toomas", "Jaana"]
mixed_elements = [1, "two", 3.14]
nested_list = [[1, 2, 3], [4, 5, 6]]

Ligipääs elementidele

Sarnaselt sõnega kasutame lõiget, et saada kätte vajalikud elemendid. Indeksid algavad 0 ja viimane element on 1 võrra väiksem kui järjendi pikkus.

numbers = [1, 2, 3]
print(numbers[0])  # 1
print(numbers[2])  # 3
print(numbers[3])  # IndexError

Kantsulgudes negatiivne number alustab otsimist järjendi tagumisest otsast. -1 on viimane element.

Elemendi muutmine

Selleks,et muuta järjendi elementi, pöördume konkreetse elemendi poole ja omistame sinna uue väärtuse.

numbers = [1, 2, 3]

print(numbers[0])  # 1

numbers[0] = 5
print(numbers[0])  # 5

Järjendi lõige

Nii nagu sõne puhul saame kasutada lõiget järjendi puhul. Erinevalt sõnest saame selle abil järjendis elemente muuta. Lõikes saab määrata alguse, lõpu ja sammu [start:end:step]. Indeksid algavad 0'st. Vaikimisi on lõike algus 0, lõpp loendi pikkus ja samm 1. Negatiivne arv otsib väärtust tagant poolt alates, sammu puhul esitab loendi tagurpidi.

# indices: 0 1 2 3 4
numbers = [1, 2, 3, 4, 5]

print(numbers[0:3])  # [1, 2, 3]
print(numbers[:3])  # [1, 2, 3]
print(numbers[2:5])  # [3, 4, 5]
print(numbers[2:])  # [3, 4, 5]
print(numbers[:])  # [1, 2, 3, 4, 5]
print(numbers[-2:])  # [4, 5]

print(numbers[0:3:2])  # [1, 3]
print(numbers[::3])  # [1, 4]
print(numbers[::-1])  # [5, 4, 3, 2, 1]
print(numbers[::-2])  # [5, 3, 1]
print(numbers[-1:0:-2])  # [5, 3]

Elementide muutmine lõiguga

numbers = [1, 2, 3, 4]
numbers[1:3] = [10, 11]
print(numbers)  # [1, 10, 11, 4]

numbers[1:3] = [0]
print(numbers)  # [1, 0, 4]

Indeks mitmemõõtmelises järjendis

matrix = [[1, 2, 3], [4, 5, 6]]

print(matrix[0])  # [1, 2, 3]
print(matrix[0][0])  # 1
print(matrix[-1][-1])  # 6
print(matrix[0][:2])  # [1, 2]

Veel järjendi muutmise töövahendeid

  • .insert(asukoht, väärtus) lisab väärtuse määratud kohale järjendis
  • .remove(väärtus) eemaldab järjendist esimese (vasakpoolse) määratud väärtusega elemendi
  • .pop(asukoht) eemaldab ja tagastab väärtuse määratud asukohas
  • .append(element) lisab elemendi järjendi lõppu
  • .extend(teine_järjend) lisab järjendi lõppu teise järjendi
  • + operaatori abil saab ühendada kaks järjendit
  • * operaatori abil saab korrata järjendi sisu

Veel töövahendeid järjendi töötlemiseks

  • .index(väärtus) tagastab esimese (vasakpoolse) määratud väärtusega elemendi indeksi
  • .count(väärtus) loendab mitu korda esineb loendis määratud väärtusega elemente ja tagastab arvu
  • in abil saab kontrollida, kas element on järjendis tagastab tõeväärtuse 1 in numbers  # True / False
  • min(järjend) tagastab kõige väiksema väärtuse järjendis
  • max(järjend) tagastab kõige suurema väärtuse järjendis
  • sum(järjend) tagastab järjendi elementide summa
  • sorted(järjend) tagastab uue järjestatud järjendi
  • järjend.sort() järjestab järjendi elemendid
