Minu bitcoin kalkulaator



Bitcoin kalkulaator teeb rehkenduse kasutaja sisendi järgi ning väljastab sobivas valuutas koguse väärtuse.
Programm küsib kasutajalt palju coine on ning mis valuutas väärtus väljastada. Peale seda teeb see päringu
bitcoin API lehelt COINDESK kust see saab coini hetke väärtused.
