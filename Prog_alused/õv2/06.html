Tsükkel (kordused)

Koodi kordamine

Tihti on vaja koodis mingit tegevust korrata. Selle asemel, et kopeerida sama koodi mitu korda, kasutame kordust. Tavapäraselt määratakse tsükliga korduste arv või lõpetamistingimus. Pythonis on kahte tüüpi kordust: while ja for

while tsükkel

Tsükli päises kirjeldatakse ära tingimus. Kui tingimus on tõene, pannakse tsükli keha (sisu) käima. Kui tingimus pole tõene, lõpetatakse tsükkel ära.

count = 0

while count < 4:
print(count)
count += 1 Tulemus: 0 1 2 3

for tsükkel

Tsükli päises kasutatakse hulka (näiteks list). Iga kordusega võetakse üks element hulgast. Antud element pannakse määratud muutujasse. Kui elemendid saavad otsa, lõpetatakse tsükkel.

for element in [0, 1, 2, 3]: print(element) Tulemus: 0, 1, 2, 3

range() funktsioon

Kasutatakse arvude genereerimiseks.

  • range(10) genereerib arvud 0-st 9-ni (kokku 10 arvu)
  • Funktsiooniga saab määrata alguse, lõpu ja sammu: range(start, stop, step)
    • start on kaasa arvatud
    • stop ei ole kaasa arvatud
  • Ühe argumendiga määratakse lõpp (algus on vaikimisi 0)

Kuna range() arve ei hoita mälus, siis nende nägemiseks tuleks kasutada funktsiooni list(), et luua nendest järjend.

list(range(3)) # => [0, 1, 2]

Funktsiooni len() abil saame kalkuleerida listi pikkust ning tsükli muutujaga koos saame listi indeksid.

names = ["Harry", "Hermione", "Ron"]

for i in range(len(names)): print(f"Element at index {i} is {names[i]}")

enumerate() funktsioon

võimaldab tsükli päises saada nii indeks kui ka element ise.

names = ["Harry", "Hermione", "Ron"]

for i, name in enumerate(names): print(f"Element at index {i} is {name}")

Tsükli katkestamine

break lõpetab tsükli täitmise. Saab kasutada olukorras, kus edasi pole tsüklit mõtet jooksutada.

names = ["Frodo", "Merry", "Pippin", "Sam"]

for name in names: print(f"Analyzing name {name}") if name.startswith("M"): print(f"Found a name starting with M: {name}")
break

else tsüklis

Tsükli lõpus else käivitub juhul, kui tsükli jooksul ei kasutatud break-i. break puudumisel koodis, toimub else juhul kui tsükkel lõppu jõuab.

Tsükli jätkamine

continue täidab tsükli järgmise sammu.

numbers = [1, 2, 4, 5, 6, 9, 10]

for nr in numbers:
if nr % 2 == 0: continue print(f"Odd number: {nr}") Tulemus: Odd number: 1; Odd number: 5; Odd number: 9

Kokkuvõte

  • Tsükliga on võimalik koodi korrata
  • Pythonis on kaks tsüklit: while ja for
  • for tsükliga saab võtta elemente andmehulgast (näiteks list)
  • while tsükliga sõltub kordamine tingimusest
  • break võimaldab tsükli keskel korduse lõpetada
  • range() genereerib täisarve, hea kasutada for tsüklis
  • enumerate() annab lisaks elemendile ka elemendi indeksi
