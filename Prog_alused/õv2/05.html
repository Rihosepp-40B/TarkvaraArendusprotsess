Tingimuslause

Hargnemine koodis

Tavapäraselt toimub koodi käivitamine järjest ridade haaval. Mõne osa koodist tahaks käivitada vaid teatud tingimusel.

Näiteks: kui piim on otsas, telli uus piim. Või: kui muru on pikk ja on soe, tuleb muru niita.

Selleks kasutame tingimuslauset: if

print("start")
temperature = 24
if temperature > 18:
print("Mow the lawn") print("done")

Kui temperatuur on üle 18 kraadi, toimub muru niitmine. Tingimuslausele järgnev taandega rida (print) täidetakse vaid juhul, kui tingimus on tõene.

Tingimuslause reale võib järgneda ka mitu tandega rida. Need moodustavad ploki, need täidetakse tervikuna järjekorras.

Osa taandeplokist võib olla tingimuslause.

print("start")
temperature = 24
if temperature > 18:
print("Mow the lawn")
print("Walk the dog")
if temperature > 21:
print("Go swimming") print("done")

Mitu valikut

temperature = 24
if temperature > 21:
print("Go swimming") elif temperature > 18:
print("Walk the dog")
print("Mow the lawn") else:
print("Make fire")

  • Tingimusi kontrollitakse järjekorras (ülevalt alla)
  • Mitu sama taandega if'i', on üks haru, kõik käivitatakse kui tingimused on True
  • Ühe if-lause ahelas (if, elif, else) vaid üks harudest pannakse käima
  • elif haru käivitub juhul kui eelnevad tingimuslaused said tulemuse False
  • else haru käivitub, kui ükski muu haru eelnevalt pole käivitunud
  • else haru pole kohustuslik
  • if elif tingimuste seadmisel tuleb jälgida nende järjekorda

if tingimused

  • Tingimusena saab kasutada avaldist, mis annab tulemuseks True või False
  • Sõnede võrdlemiseks kasutatakse ==
  • Saab luua muutujaid, milles hoitakse tõeväärtust: is_warm = temperature > 18

Loogilised avaldised

Keerukamas olukorras on vaja kasutada mitut tingimust. Selleks kasutatakse loogilisi tehteid:

  • and: tõene siis kui tingimuse mõlemad osad on tõesed (is_warm and is_raining)
  • or: tõene siis kui üks tingimuse osadest on tõene (is_cold or is_raining)
  • not: tõene siis kui tingimus on väär (not is_raining)

Tingimuslause

  • Tingimuslausega saab valikuliselt koodikäivitamist kontrollida
  • Tingimuslausel võib olla mitu haru, millest maksimaalselt 1 täidetakse
  • Haru käivitamist saab kontrollida tingimusega
  • Tingimus võib olla keerukam loogika avaldis
