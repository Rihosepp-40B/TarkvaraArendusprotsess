Positsioonilised arvusüsteemid

Kasutame igapäevaselt arvude väljendamiseks kümnendsüsteemi (x10). See tähendab, et kõik arvud esitatakse kümne erineva numbriga (0, 1, ..., 9).

Kahendsüsteemi (x2) ehk binaarsüsteem peamiseks kasutusalaks on arvutid. Kahendsüsteemis esitatakse arve sarnaselt kümnendsüsteemiga. Vahe on selles, et kümne numbrimärgi asemel kasutatakse kahte: 0 ja 1. Kohakaalud on kahe astmed (ehk siis kahendsüsteemi puhul on alus 2).

Kahendsüsteem on nii ölda masinkeel (1 ja 0, tõene ja väär). Neid arve võib kohati olla keeruline lugeda, selleks on tehtud lihtsustamiseks kuueteistkümnend (x16) süsteem.

Teisendamised: x10 ⇋ x2 ⇋ x16

Kümnendarvu saamiseks binaar numbriks on kaks võimaliku meetodit:

  1. Kümnend arvu jagamine kahega ilma komadeta.

    2. Osa mida ei saa jagada ilma komadeta on jääk. Jääk = 1, ilma jäägita = 0. Jäägita tulemus jagatakse uuesti kahega ja nii edasi kuni jõutakse tehteni 1 jagada 2, mis annab jäägi 1. Seejärel pannakse numbrid kokku alustades viimasest jäägist ning nii saadakse binaarnumber.

    Arv // 2 = tulemus Arv % 2 = jääk
    376 // 2 = 188 0
    188 // 2 = 94 0
    94 // 2 = 47 0
    47 // 2 = 23 1
    23 // 2 = 11 1
    11 // 2 = 5 1
    5 // 2 = 2 1
    2 // 2 = 1 0
    1 // 2 = 0 1

    Lugedes jääki alt üles saame binaarnumbri: 101111000

  2. Tabelis visuaalse võrdluse järgi.

    3. See meetod vajab võrdlus tabelit. Tabeli võrdlus kohaks on kümnendarvud mis tähistavad binaarnumbri väärtust antud kohal.

    Kümnendarvu sisestatakse tabeli vasakust lahtrist. Kui number on võrdne või suurem siis antakse väärtus 1, väiksema puhul 0. Kui number oli suurem või võrnde, siis lahutatakse numbrist tabelis olev arv ja jäägiga liigutakse tabelis edasi. Väärtuse 0 puhul liigub kogu väärtus edasi.

    binaar arvu koha väärtus 512 256 128 64 32 16 8 4 2 1
    Sisend -> 376 376 376 120 120 56 24 8 0 0 0
    kui suurem või =, siis 1 0 1 0 1 1 1 1 0 0 0

    Lugedes tabeli alumist rida vasakult paremale saame binaarnumbri: 101111000

Kahendsüsteemi teisendamiseks kümnendsüteemi võetakse binaarnumbri iga number, mis korrutatakse kahega, mis on astendatud tema positsiooniga (positsioonid algavad nullist) ning kõik need arvud liidetakse kokku

Näide: Meil on kahendsüsteemi arv 10111100
Arvutus: 1*2^8 + 0*2^7 + 1*2^6 + 1*2^5 + 1*2^4 + 1*2^3 + 0*2^2 + 0*2^1 + 0*2^0 = 256 + 0 + 64 + 32 + 16 + 8 + 0 + 0 + 0 = 376

Kuueteistkümnend süsteemi teisendamist tehakse sarnaselt nagu tehti seda kahendsüsteemiga, aga kaks asendatakse kuueteistkümnega.
