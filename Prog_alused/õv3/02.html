Pärimine

Pärilikkus (ingl k inheritance) on ühe klassi (laps(child) ka alamklass) võime tuletada või pärida omadused teiselt klassilt (vanem (parent) ka ülemklass). Laps klass omandab kõik vanema klassi omadused, võib muuta omandatud väärtust (polümorfism) ning lisab täiendavaid omadusi (laiendab (extends)).

class Parent(): def greet(self): print("Hello")
class Child(Parent): def jumparound(self): print("*Jump-jump-jump*")
Vanem klass omab funktsiooni greet, laps klass pärib selle omaduse ning lisab funktsiooni jumparound

vanem = Parent()  # loome objekti vanem
laps = Child()  # loome objekti laps

vanem.greet()  # prindib "Hello"
laps.greet()  # prindib "Hello", kuna pärsi selle omaduse
vanem.jumparound()  # AttributeError: kuna sellel objektil ei ole seda omadust
laps.jumparound()  # prindib "*Jump-jump-jump*"
