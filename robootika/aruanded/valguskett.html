Valguskett

Uuritud funktsioonid

  • sizeof() - mõõdab paljui mälu kasutab muutuja või andmed. sellel harjutusel kasutatud selleks, arvutada välja järjendi pikkus sizeof(led) / sizeof(led[0])
  • if (loogika) - tingimuse kontroll, kui loogika on tõene, siis { tee midagi } else { tee midagi muud}.
  • analogRead(A0) - analoogse seadme lugemine, sulgudes väärtus viitab pesale kuhu on ühendatud.
  • map(mida, väärtus1, väärtus2, uus1, uus2) - selle abil saab leida mingid väärtuste vahemiku ja asendad need uutega.
  • constrain(muutuja, 0, 4) - piirab numbrid olema määratud vahemikus.

Töö kirjeldus

Vaja luua valguskett 5 erineva reziimiga.

Kasutatud komponendid

  • Arduino UNO plaat - 1tk
  • Arendusplaat - 1tk
  • Juhtmed - 17tk
  • Takisti - 6tk (220Ω)
  • Potentsiomeeter - 1tk
  • LED roheline - 2tk
  • LED kollane - 2tk
  • LED punane - 2tk

Töö protsess

Kood Tinkercad'is valmis ja testitud. Ühendasin dioodid, takistid, potentsiomeetri ja juhtmed. Ühendasin Arduino plaadi arvutiga ning laadisin progammi üles.

Tulede reziime kontrollib potentsiomeeter, mille skaala väärtustele on antud uued tähendused (0 kuni 4).

  • 1.režiim - kõik tuled koos lülituvad sisse ja välja.
  • 2.režiim - üks tuli korraga sisse välja.
  • 3.režiim - kõik ühtevärvi tuled sisse ja välja.
  • 4.režiim - küünlaefekt, juhuslikud tuled sisse välja.
  • 5.režiim - valgus jookseb läbi.
