Sõnastik

Kui järjendil on elemendid järjestikus arvudega positsioonil, siis sõnastikus (dict) kasutatakse positsiooniks võtit (key). Võti võib olla nii sõne kui ka arv (kõik muutumatud andmetüübid sobivad). Võtme abil leitakse sõnastikust üles kaasnevad väärtused (value).

Väärtused võivad olla lisaks sõnedele ja numbritele omakorda järjendid või sõnastikud. Ühe võtme kohta on üks väärtus (ei saa olla mitu sõne, järjendit jms ühe võtme all).

Sõnastiku loomine

Sõnastiku loomiseks kasutatakse loogelisi sulge {} (kasutatakse ka enniku (tuple) loomiseks). Lisaks on võimalik kasutada käsku dict(). Sõnastikus nimetatakse võti ja sellele omistatakse väärtus. Koos on need üks element (koosnevad ennikust (tuple)). Esimesel kohal on alati võti, teisel väärtus.

a = dict(one=1, two=2, three=3)
b = dict([('two', 2), ('three', 3), ('one', 1)])
d = {'three': 3, 'one': 1, 'two': 2}
tühi_sõnastik = {}

Töövahendid sõnastikuga

  • len(sõnastik) sõnastiku pikkuse saamiseks (loeb võtmed kokku).
  • dict["võti"] konkreetse elemendi poole pöördumine.
  • dict["võti"] = "väärtus" olemas oleva võtme väärtuse muutmine või uue võtme loomine.

print(d['one'])  # => 1
d['one'] = 5  # value for key 'one' will become 5

  • key in dict kas selline võti on olemas (tagastab tõene või väär).
  • dict.keys() tagastab võtmete nimekirja.
  • dict.values() tagastab väärtuste nimekirja.
  • dict.items() tagastab võtme ja väärtuse paaride nimekirja.
  • for tsükkel:

võti-väärtus paarid
for name, phonenr in phones.items(): print(name, phonenr) tsükkel üle sõnastiku annab kõik võtmed:
for name in phones: print(name, phones[name])
