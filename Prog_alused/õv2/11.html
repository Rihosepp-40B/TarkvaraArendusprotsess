Hulk

Hulk (set) on järjestamata elementide kollektsioon. Selles ei ole kordusi. Elemente ei saa indeksi järgi pärida. Hulga elemendid peavad olema muutumatud (immutable). Ta ise on muutuv (elemente saab lisada, eemaldada). Hulgaga saab teostada matemaatilisi hulga operatsioone.

Hulga loomine

Hulga tähistamiseks kasutatakse loogelisi sulge {}, kuid tühja hulga saab luua ainult korraldusega set(). Selle abil saab luua hulka ka olemasolevatest järjenditest (hulka jäävad ainult unikaalsed väärtused, korduvaid väärtusi pole).

my_set = {1, 2, 3} # => {1, 2, 3}
empty_set = set() # => set()

numbers = [1, 2, 3, 1]
numbers_set = set(numbers) # => {1, 2, 3}

Hulka ei saa panna muutuvaid objekte. See annab vea: "TypeError: unhashable type: 'list'". List võib programmi jooksul muutuda, seepärast seda hulgas ei saa kasutada. Kasutada saab: sõne, arvud, tõeväärtus, ennik, None jt.

Töövahendid hulgaga

  • hulk.add(väärtus) elemendi lisamine.
  • hulk.update([väärtus1, väärtus2]) mitme elemendi lisamine.
  • hulk.discard(väärtus) elemendi eemaldamine, ignoreerib puuduvat väärtust.
  • hulk.remove(väärtus) elemendi eemaldamine, annab vea teate puuduva väärtuse korral
  • hulk.pop() suvalise elemndi eemaldamine ja tagastamine, kuna hulga puhul ei ole indeksit, siis võtab see korraldus suvalise elemendi.
  • for tsükliga saab elemendid kätte.

numbers = {4, 1, 6}
for number in numbers: print(number)  # => 1, 4, 6 Saadavate elementide järjekord ei ole garanteeritud.

  • hulk_a.union(hulk_b) ühend - kõik elemendid, mis kuuluvad ühte või teise hulka või mõlemasse. Saab ka nii: hulk_a | hulk_b
  • hulk_a.intersection(hulk_b) ühisosa - kõik elemendid, mis kuuluvad mõlemasse hulka. Saab ka nii: hulk_a & hulk_b
  • hulk_a.difference(hulk_b) vahe - kõik elemendid, mis kuuluvad esimesse hulka, aga ei kuulu teise hulka. Saab ka nii: hulk_a - hulk_b
  • hulk_a.symmetric_difference(hulk_b) sümmeetriline vahe - kõik elemendid, mis kuuluvad esimesse või teise hulka, aga ei kuulu mõlemasse. Saab ka nii: hulk_a ^ hulk_b
  • len(hulk) hulga pikkus.
  • Sõne tähed hulgaks: my_set = set("hello") annab hulga {'l', 'o', 'e', 'h'}
  • Kas element on hulgas: 'h' in my_set annab True

a = {1, 2, 3} b = {2, 3, 4, 5} print(a | b)  # {1, 2, 3, 4, 5} print(a & b) # {2, 3} print(a - b) # {1} print(a ^ b) # {1, 4, 5}
